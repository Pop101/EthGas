Historical Gas API

An API for accessing Ethereum gas prices from the past year.
Currently, the API is not rate-limited and updates once every five seconds. It's uptime is not 100%, so don't rely on it.

Endpoints

  • /gas
  • /start

Gas

Get the current low, average, and high gas costs

Example Response:
{
  "status": 1,
  "message": "OK",
  "result": {
    "11554042": {
      "avg": 51.0,
      "date": 1609313299,
      "high": 123.0,
      "low": 43.0
    }
  }
}

Optional Arguments:

Argument Description Type Example
num Specifies the number of results to return. Defaults to 1 integer /gas?num=5
date Specifies the most recent date returned. Defaults to the most recent value integer, UNIX timestamp /gas?date=
1609313299
block Specifies the most recent block number returned. Defaults to the most recent value and overrides date
 integer /gas?block=
11554042

Start

Gets the date and block of the oldest stored entry

Example Response:
{
  "message": "OK",
  "result": {
    "firstBlock": 11551387,
    "firstDate": 1609278395
  },
  "status": 1
}

Optional Arguments: None
